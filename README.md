# The Boating Strategic Brief

A Chrome Extension that delivers daily strategic briefings from boatingindustry.com, powered by OpenAI.

---

## What It Does

- Fetches news from boatingindustry.com every day at **8 AM**
- Filters news by **strategic importance** (Financial, Technology, Dealer/Retail)
- Shows an **executive summary** with the most important industry developments
- **Removes duplicates** so you don't see the same news twice

---

## Installation

### Step 1: Download

1. Click the green **"Code"** green button above
2. Click **"Download ZIP"**
3. Unzip the folder on your computer

### Step 2: Load in Chrome

1. Open Chrome and go to `chrome://extensions`
2. Turn on **"Developer mode"** (top right toggle)
3. Click **"Load unpacked"**
4. Select the **`boating-strategic-brief`** folder

### Step 3: Add Your API Key

1. Click the extension icon (anchor) in Chrome toolbar
2. Click the **gear icon** to open Settings
3. Enter your OpenAI API Key
   - Get one at: https://platform.openai.com/api-keys
4. Click **"Save Settings"**

---

## How to Use

- **View Briefing**: Click the extension icon
- **Refresh**: Click the "Refresh" button for a new briefing
- **Test Auto-Refresh**: In Settings, click "Set Test Alarm" to verify it works

---

## Features

| Feature | Description |
|---------|-------------|
| Daily Briefings | Auto-generates at 8 AM |
| Priority Tags | HIGH / MEDIUM / LOW importance |
| Source Links | Click to read full articles |
| Deduplication | Never see the same article twice |
| Manual Refresh | Get updates anytime |
